Acerca de

Python en la Oceanografía

El uso de Python en la Oceanografía permite generar metodologías automatizadas a través de una serie de comandos y paqueterías de acceso libre. Dentro de las aplicaciones que Python brinda está el web scraping en portales con información satelital (PODAAC, ocean color), el procesamiento de datos (CTD, ADCP, muestras biológicas), generación de gráficos, análisis estadísticos, Machine Learning, entre otras.
